# MollyCaffee Bot

Простой бот для программы лояльности кофейни.

## Функционал

- Клиенты: смотреть штампы, добавлять штамп, использовать бесплатный кофе (через QR-код)
- Бариста: авторизация, сканирование QR-кодов, добавление штампов вручную
- Данные сохраняются в SQLite
